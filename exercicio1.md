Exercício 1 — Meu cartão de apresentação

Crie uma página HTML simples representando um cartão de apresentação pessoal.

Seu cartão deve conter:
seu nome usando a tag <h1>
sua profissão ou área de interesse usando a tag <p>
uma pequena descrição com 2 ou 3 linhas
um botão com o texto "Entrar em contato"

Depois, crie um arquivo CSS externo e estilize os elementos.

Requisitos:
Estrutura HTML

Utilize:
<main>
<section>
<h1>
<p>
<button>
Estilização CSS

Seu CSS deve utilizar:

Seletores:
seletor por tag
seletor por classe


Tipografia:
font-family
font-size
font-weight

Cores:
cor de fundo
cor de texto

Botão:
cursor
transition
Interação:

Criar efeito com :hover quando o mouse passar sobre o botão.


Exercício 2 — Lista de tecnologias favoritas

Crie uma página HTML com uma lista das tecnologias que você conhece ou gostaria de aprender.

Sua página deve conter:

um título principal
um subtítulo
uma lista com pelo menos 6 tecnologias
uma breve descrição abaixo da lista

Depois, estilize com CSS externo.

Requisitos:
Estrutura HTML
<main>
<h1>
<h2>
<ul>
<li>
<p>

CSS
Agrupamento de seletores
Aplique mais de uma classe em pelo menos 2 elementos.
Aplicar transparência em algum elemento.
Pseudo-elementos: Adicionar um símbolo antes de cada tecnologia.

Exemplo:
estrela
check
foguete


Exercício 3 — Analisando e recriando uma interface

Observe a imagem disponibilizada. Seu objetivo será:

Parte 1 — Análise

Observe cuidadosamente a interface e responda:

- Quais elementos HTML você acredita que foram utilizados?
- Quais propriedades CSS você acredita que foram utilizadas?

Parte 2 — Implementação

Com base na sua análise, recrie a estrutura HTML e implemente o CSS da forma mais próxima possível

Bônus
Pesquisar como utilizar:
::before
opacity
box-shadow

para deixar a interface mais próxima da original.
